A css design tool for PyQt.

Objects you can design in the program:
    QLabel
    QPushButton
    QLineEdit
    QTextEdit
    QRadioButton
    QCheckBox
    QComboBox
    QSpinBox
    QSlider
    QListWidget
    QGroupBox
    QTabWidget
    QDockWidget
    QMenu
    QScrollBar
You can design these objects and more like "hover, pressed, marked, indicator, close-button, tab-bar, title".
